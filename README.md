

## Architecture
- **Pattern:** Provider + Dashboard UI
- **State management:** Provider (ChangeNotifier)
- **Calculator type:** Scientific (Bonus)
- **Color theme:** ocean_dark

## Features

- Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Sign toggle (+/−)
- Backspace / clear
- Trigonometric: sin, cos, tan, asin, acos, atan (degrees mode)
- Hyperbolic: sinh, cosh, tanh
- Logarithms: log₁₀, ln
- Power: x², x³, 1/x, |x|, √x
- Constants: π, e
- Factorial: n!
- Combinatorics: nPr, nCr
